# Estatly Mobile App

The official mobile application for Estatly clients, built with React Native and Expo. Experience premium real estate browsing and management on the go.

## 📱 Features

- **Property Feed:** Scroll through beautiful imagery of available listings.
- **Saved Properties:** Wishlist homes for future consideration.
- **Viewing Requests:** Book showings directly from the app.
- **Offer Management:** Submit and track offers on properties.
- **Real-Time Chat:** Message your agent instantly.
- **Push Notifications:** Stay updated on price drops and new listings.

## 🛠 Tech Stack

- **React Native** & **Expo**
- **TypeScript**
- **React Navigation**
- **Zustand** (State Management)

## 🚀 Setup & Run

1. Install dependencies:
```bash
npm install
```

2. Start the Metro bundler:
```bash
npm start
```

3. Scan the generated QR code using the **Expo Go** app on your iOS or Android device.
