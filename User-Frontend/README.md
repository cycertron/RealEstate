# Estatly Public Portal (User Frontend)

The public-facing website for the Estatly Real Estate management platform. Browse premium properties, search availability, and connect with agents using a modern, responsive React TypeScript interface.

## 🎯 Features

- 🏠 **Property Browsing** - Discover available luxury villas, apartments, and commercial spaces.
- 🔍 **Search & Filter** - Find perfect properties by location, type, price, and amenities.
- 📅 **Viewing Scheduling** - Easy appointment requests for showings.
- 👤 **Client Accounts** - Sign up, login, and manage your property interests.
- 💬 **Agent Messaging** - Direct communication with assigned real estate agents.
- ⭐ **Agency Reviews** - See client testimonials and ratings.
- 📱 **Responsive Design** - Mobile-optimized interface with premium dark-mode aesthetics.

## 💻 Technology Stack

- **React 19** with TypeScript
- **Vite 8** for lightning-fast builds
- **Tailwind CSS 3.4** for utility-first styling
- **React Router v6** for client-side routing
- **Framer Motion** for micro-animations and smooth transitions

## 🚀 Quick Start

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The application will typically start on `http://localhost:5174`.
